    about_TestDataGeneration

SHORT DESCRIPTION
    PowerShell module for data / code generation.

LONG DESCRIPTION
    Provides utility commands and classes for data and code generation.

EXPORTED COMMANDS
    Following is a list of commands exported by this module:

    Get-RandomCharacterSource
        Get characters for random string generation.

    Convert-RangePatternToTuple
        Converts range pattern strings to tuple classes.

    Test-RangePatternString
        Tests validity of a range pattern string.

    Get-RandomInteger
        Generates random integer values.

    Select-Random
        Randomly selects values(s) from input values.

    Get-RandomIpV4Address
        Generates random IPv4 address strings.

    Get-RandomString
        Generates random string values.
