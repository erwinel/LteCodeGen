SHORT DESCRIPTION
    PowerShell module for data / code generation.

LONG DESCRIPTION
    Provides utility commands and classes for data and code generation.

EXPORTED COMMANDS
    Following is a list of commands exported by this module:

    Get-RandomCharacterSource
        Get characters for random string generation.

    Convert-RangePatternToRandomInt
        Converts range pattern strings to random integers.

    ConvertFrom-BinaryNotation
        Converts binary notation strings to number values.

    ConvertTo-BinaryNotation
        Converts number values to binary notation strings.

    Get-RandomInteger
        Generates random integer values.

    Remove-NonPrintableChars
        Removes non-printable characters from strings.

    Select-Random
        Randomly selects values(s) from input values.

    Get-RandomIpV4Address
        Generates random IPv4 address strings.
